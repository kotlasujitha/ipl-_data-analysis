# IPL Data Analysis 🏏📊
## About
Analysis of 16 seasons of IPL data using Python.
## Key Findings
- MI and CSK dominate with most wins
- Winning toss only helps 50.6% of the time
- Total runs per season have grown every year
- Virat Kohli is the all-time top run scorer
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
## Dataset
Source: Kaggle — IPL Complete Dataset
